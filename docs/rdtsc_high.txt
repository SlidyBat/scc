__rdtsc_high
============
:title: Shellcode Compiler Runtime

---------------------------------------------
uint32_t __rdtsc_high();
---------------------------------------------

Reads the time stamp counter on the x86 architecture.  Returns the upper 32 bits of the time stamp counter.
